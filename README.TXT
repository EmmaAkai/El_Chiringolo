Proyecto Pre-Integrador - Página Web HTML y CSS
Este es un proyecto pre-integrador desarrollado como práctica para Talento Tech. La página fue construida únicamente con HTML y CSS e incluye diversas secciones que muestran la implementación de diferentes técnicas de diseño y adaptabilidad.

Características del Proyecto

Contiene la estructura principal de la página y da la bienvenida a los usuarios.
Sección de Productos

Diseñada usando Flexbox para organizar los elementos de manera responsiva y optimizada en diferentes dispositivos.
Sección de Contacto

Incluye un formulario de contacto funcional, implementado a través de Formspree para el envío de mensajes de usuarios sin necesidad de backend.
Adaptabilidad

Se aplicaron media queries para asegurar que el diseño se adapte a diferentes tamaños de pantalla, brindando una experiencia amigable en dispositivos móviles, tabletas y escritorio.
Sección de Reseñas

Una sección inventada que presenta comentarios de clientes ficticios. Esta sección fue diseñada utilizando CSS Grid, logrando una disposición ordenada y moderna.
Tecnologías Utilizadas
HTML5: para la estructura y contenido de la página.
CSS3: para el diseño, estilo, y adaptabilidad de las diferentes secciones.
Formspree: para la funcionalidad de envío de datos en la sección de contacto.

Implementación de Carrito de Compras
En una iteración posterior del proyecto, se implementó un carrito de compras completamente funcional con las siguientes características:

Almacenamiento de Productos:
Los productos seleccionados por el usuario se almacenan en cantidades específicas utilizando sessionStorage. Esto permite que la selección de productos persista durante la sesión del usuario.

Confirmación de Compra:
Se desarrolló una nueva pestaña de confirmación de compras. En esta página, el carrito de compras muestra:

Todos los productos seleccionados.
La cantidad de cada producto.
El valor acumulado de la compra, mostrando el total de forma clara y concisa.
Página de Agradecimiento:
La página de confirmación incluye un botón final de "Comprar". Al presionarlo, se redirige al usuario a una nueva pestaña de agradecimiento, que destaca un mensaje cálido por confiar en la página web.

Funcionalidades Técnicas Adicionales
Interactividad Dinámica:
La implementación del carrito de compras y el manejo de datos en la página de confirmación se logró mediante el uso de JavaScript para una experiencia más interactiva.

Persistencia Temporal:
El uso de sessionStorage asegura que los datos del carrito sean temporales y se eliminen al cerrar la pestaña, manteniendo la privacidad del usuario y evitando datos residuales.

Futuras Mejoras
Este proyecto se puede extender con funcionalidades adicionales como:

Incorporar autenticación de usuarios para compras personalizadas.
Integrar métodos de pago reales a través de servicios como PayPal o Stripe.
Crear un backend para almacenar datos de compras y usuarios de manera más permanente.
